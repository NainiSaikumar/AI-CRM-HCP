# AI CRM HCP

## Overview

AI CRM HCP is a full-stack Healthcare Customer Relationship Management (CRM) application that helps medical representatives manage healthcare professional (HCP) interactions efficiently. The application also uses AI to generate interaction summaries, sentiment analysis, and follow-up recommendations.

---

## Features

- Log HCP interactions
- View interaction history
- AI-generated interaction summary
- Sentiment analysis
- Follow-up recommendations
- Dashboard analytics
- Search interactions
- Filter by sentiment
- Export interactions as PDF

## Installation

### Frontend

```bash
cd frontend

npm install

npm run dev
```

### Backend

```bash
cd backend

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

python -m uvicorn app.main:app --reload
```
